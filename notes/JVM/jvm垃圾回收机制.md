### JVM调优

#### 参数

|        -Xms         | 设置初始值分配大小，默认为               |
| :-----------------: | ---------------------------------------- |
|        -Xms         | 设置初始分配大小，默认为物理内存的"1/64" |
|        -Xmx         | 最大分配内存，默认为物理内存的"1/4"      |
| -XX:+PrintGCDetails | 输出详细的GC处理日志                     |

具体使用方法：

```java
-Xms1024m -Xmx1024m -XX:+PrintGCDetails
```

#### GC是什么

- 频繁收集Young区
- 较少收集Old区
- 基本不动Perm区
